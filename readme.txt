=== Description ===
TOGETHER is a wedding WordPress theme which specially designed for marriage, engagement, personal, wedding websites. It is optimized and targeted to create clean and beautiful wedding websites and blogs quickly and easily. Theme included theme customizer for live editing and easy to manage homepage through home page builder. It includes color schemes, header and footer option, custom banner images, social profiles and more features for your ease to create elegant websites. Please find more details about theme at http://daisythemes.com/theme/together/ and Demo at http://demo.daisythemes.com/together/


You can get support in http://daisythemes.com/support and Details about theme at http://daisythemes.com/theme/together/ and Demo at http://demo.daisythemes.com/together/
